 Error 412 - Tus.io Precondition Failed

 For tus.io endpoints, the tus.io standard requires the HTTP header 'Tus-Resumable' to be set to a valid tus.io version.
 The only version supported by the Ember Nexus API is '1.0.0'.



 Link to the documentation: https://ember-nexus.github.io/api

 Link to http.dev's explanation of the 412 error code: https://http.dev/412
